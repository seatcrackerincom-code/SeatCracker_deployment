STEP 1 — CALCULATE MARKS (JEE Advanced official marking scheme)
================================================================

Apply this scheme to EVERY question in BOTH papers:

SINGLE CORRECT MCQ:
  - Correct answer chosen     = +3
  - Wrong answer chosen       = -1
  - Not attempted (null)      = 0

MULTI CORRECT MCQ:
  - All correct options chosen and no wrong option = +4 (full marks)
  - Only some correct options chosen, no wrong option chosen:
      = +1 for each correct option chosen (partial credit, max +4)
  - Any wrong option chosen (even with correct ones) = -2
  - Not attempted (null) = 0

NUMERICAL ANSWER TYPE:
  - Correct value (exact match or within allowed range) = +4
  - Wrong value = 0
  - Not attempted (null) = 0

Calculate separately for Paper 1 and Paper 2:
  - physics_score, chemistry_score, maths_score
  - paper_total (max 180 per paper)
  - correct_count, wrong_count, unattempted_count (overall + per subject)

Then calculate COMBINED (Paper 1 + Paper 2):
  - total_score (max 360)
  - physics_total = paper1_physics + paper2_physics (max 120)
  - chemistry_total = paper1_chemistry + paper2_chemistry (max 120)
  - maths_total = paper1_maths + paper2_maths (max 120)
  - overall_accuracy = (total_correct / (total_correct + total_wrong)) * 100
  - time_efficiency_tag: if unattempted > 30% = "Too many skipped",
    if wrong > 40% of attempted = "Accuracy needs work",
    else = "Good attempt balance"

================================================================
STEP 2 — CALCULATE PERCENTILE AND RANK
================================================================

Use this formula (NTA standard method):

  percentile = (students_scoring_equal_or_below / total_students) * 100

Since we don't have real data of all students, estimate using
a realistic bell curve distribution for JEE Advanced:
  - Assume average score on platform = 35% of 360 = ~126 marks
  - Assume standard deviation = 55 marks
  - Use: students_below = total_students * normalCDF(student_score, 126, 55)
  - normalCDF approximation:
      z = (score - 126) / 55
      cdf ≈ 0.5 * (1 + erf(z / sqrt(2)))
      erf(x) ≈ 1 - (1/(1 + 0.2316419*|x|))^5 * 0.3989423 * e^(-x^2/2) * 1.330274

  percentile = (students_below / total_students) * 100
  rank_estimate = total_students - round(students_below) + 1

Round percentile to exactly 2 decimal places.
Round rank_estimate to nearest integer.

Also calculate SUBJECT-WISE PERCENTILE for Physics, Chemistry, Maths
individually using the same method with subject avg = 42, std_dev = 18.

Check qualification for every category using COMBINED score.

Category thresholds (approximate, based on 2024 JEE Advanced):

  GEN (CRL)  : aggregate >= 10% of 360 = 36 marks

For each category output:

HOW TO CALL THIS IN SEATCRACKER (JavaScript example):
================================================================

const SYSTEM_PROMPT = `<paste entire prompt above here>`;

async function getResults(day, paper1Answers, paper2Answers, totalStudents) {
  const response = await fetch("https://api.anthropic.com/v1/messages", {
    method: "POST",
    headers: { "Content-Type": "application/json" },
    body: JSON.stringify({
      model: "claude-sonnet-4-20250514",
      max_tokens: 1000,
      system: SYSTEM_PROMPT,
      messages: [{
        role: "user",
        content: JSON.stringify({
          day: day,
          total_students_on_platform: totalStudents,
          paper1: { answers: paper1Answers },
          paper2: { answers: paper2Answers }
        })
      }]
    })
  });

  const data = await response.json();
  const result = JSON.parse(data.content[0].text);
  return result;
}
