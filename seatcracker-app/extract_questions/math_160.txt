∑ SETS, RELATIONS & FUNCTIONS
Q001.
If A = {1,2,3,4} and B = {3,4,5,6}, then A I B (symmetric difference) is:
A) {1,2,5,6}
B) {3,4}
C) {1,2,3,4,5,6}
D) {}
I A) {1,2,5,6}
I A I B = (A∪B) – (A∩B) = {1,2,3,4,5,6} – {3,4} = {1,2,5,6}
Q002.
The number of subsets of a set with n elements is:
A) n²
B) 2n
C) 2I
D) n!
I C) 2I
I Every element has 2 choices (in or out). Total subsets = 2I. For n=3: 2³=8 subsets.
Q003.
If n(A)=3, n(B)=4 and n(A∩B)=2, then n(A∪B) is:
A) 5
B) 7
C) 9
D) 3
I A) 5
I n(A∪B) = n(A)+n(B)–n(A∩B) = 3+4–2 = 5
Q004.
A relation R on set A is called an equivalence relation if it is:
A) Reflexive and symmetric only
B) Symmetric and transitive only
C) Reflexive, symmetric and transitive
D) Anti-symmetric and transitive
I C) Reflexive, symmetric and
transitive
I Equivalence relation must satisfy all three: reflexive, symmetric AND transitive.
Q005.
Which of the following functions is one-one (injective)?
A) f(x)=x²
B) f(x)=sinx
C) f(x)=2x+3
D) f(x)=|x|
I C) f(x)=2x+3
I f(x)=2x+3 is strictly increasing → one-one. Others fail: f(x)=f(-x) for x² and |x|.
Q006.
The domain of f(x) = √(4–x²) is:
A) (–2, 2)
B) [–2, 2]
C) (0, 2)
D) [0, 4]
I B) [–2, 2]
I Need 4–x² ≥ 0 → x² ≤ 4 → –2 ≤ x ≤ 2. Endpoints included (square root of 0 is 0).
Q007.
If f(x) = x² and g(x) = √x, then fog(x) is:
A) x
B) x²
C) √x
D) xI
I A) x
I fog(x) = f(g(x)) = f(√x) = (√x)² = x, for x ≥ 0.
Q008.
The inverse function of f(x) = (x–1)/(x+1) is:
A) (x+1)/(x–1)
B) (1+x)/(1–x)
C) (x–1)/(x+1)
D) 1/f(x)
I B) (1+x)/(1–x)
I Let y=(x–1)/(x+1). Swap x,y and solve: x(y+1)=y–1 → y=(1+x)/(1–x).
∑ MATRICES & DETERMINANTS
Q009.
If A is a 3×3 matrix and |A| = 5, then |3A| is:
A) 15
B) 45
C) 135
D) 375
I C) 135
I |kA| = kI|A| for an n×n matrix. |3A| = 3³×5 = 27×5 = 135.
Q010.
A square matrix A is singular if:
A) |A| ≠ 0
B) |A| = 0
C) A = I
D) A = AI
I B) |A| = 0
I Singular matrix: |A|=0, no inverse exists. Non-singular: |A|≠0, inverse exists.
Q011.
If A = [[2,1],[5,3]], then AI¹ is:
A) [[3,–1],[–5,2]]
B) [[3,1],[5,2]]
C) [[–3,1],[5,–2]]
D) [[2,–1],[–5,3]]
I A) [[3,–1],[–5,2]]
I |A|=6–5=1. AI¹=(1/|A|)×adj(A). adj(A)=[[3,–1],[–5,2]]. Since |A|=1, AI¹=adj(A).
Q012.
The value of the determinant |1 2 3 / 4 5 6 / 7 8 9| is:
A) 0
B) 6
C) –6
D) 45
I A) 0
I R3 = R1+R2 (7=1+4? No). Actually R3–R2=R2–R1 (AP rows). Rows in AP → det=0.
Q013.
If A is symmetric, then AI equals:
A) –A
B) A
C) AI¹
D) 0
I B) A
I Symmetric matrix: AI = A. Skew-symmetric: AI = –A.
Q014.
The rank of the zero matrix (all entries = 0) is:
A) 1
B) n
C) 0
D) Undefined
I C) 0
I Rank = number of non-zero rows in row echelon form. Zero matrix has no non-zero
rows → rank 0.
Q015.
If AB = BA, then matrices A and B are said to be:
A) Orthogonal
B) Commutative
C) Idempotent
D) Involutory
I B) Commutative
I Commutative property of multiplication. Generally AB ≠ BA for matrices.
Q016.
For any square matrix A, A + AI is always:
A) Skew-symmetric
B) Symmetric
C) Zero matrix
D) Identity matrix
I B) Symmetric
I (A+AI)I = AI+A = A+AI. So it equals its own transpose → symmetric.
Q017.
The cofactor CII of matrix A = [[1,2],[3,4]] is:
A) 3
B) –3
C) 4
D) –4
I B) –3
I CII = (–1)^(1+2) × MII = (–1)³ × 3 = –3 (minor MII is the element 3).
Q018.
If A² = A, the matrix A is called:
A) Involutory
B) Nilpotent
C) Idempotent
D) Orthogonal
I C) Idempotent
I A² = A → idempotent. A² = I → involutory. AI = 0 → nilpotent. AAI = I → orthogonal.

The modulus of z = 3 + 4i is:
A) 3
B) 4
C) 5
D) 7
I C) 5
I |z| = √(3²+4²) = √(9+16) = √25 = 5. (3,4,5 Pythagorean triple)

The argument of z = –1 + i√3 is:
A) π/3
B) π/6
C) 2π/3
D) 5π/6
I C) 2π/3
I z is in 2nd quadrant. tanI¹(√3/1)=π/3. arg = π–π/3 = 2π/3.
Q021.
iII equals:
A) 1
B) –1
C) i
D) –i
I D) –i
I Powers of i cycle with period 4. 47 = 4×11+3. iII = i³ = –i.
Q022.
The conjugate of z = (2+3i)/(1–i) is:
A) (–1+5i)/2
B) (–1–5i)/2
C) (1–5i)/2
D) (1+5i)/2
I B) (–1–5i)/2
I Multiply num & denom by (1+i): (2+3i)(1+i)/2 = (2+2i+3i–3)/2 = (–1+5i)/2. Conjugate:
(–1–5i)/2.
Q023.
De Moivre's theorem: (cosθ + i sinθ)I equals:
A) cos(nθ) + i sin(nθ)
B) n cosθ + n i sinθ
C) cos(θ/n)+i sin(θ/n)
D) cosIθ + i sinIθ
I A) cos(nθ) + i sin(nθ)
I De Moivre's: (cisθ)I = cis(nθ) = cos(nθ)+i sin(nθ). Fundamental to complex number
theory.
Q024.
The cube roots of unity are 1, ω, ω². Then 1 + ω + ω² equals:
A) 1
B) 3
C) 0
D) –1
I C) 0
I Sum of nth roots of unity = 0. So 1+ω+ω²=0. Also ω³=1 and ω≠1.
Q025.
If z = r(cosθ + i sinθ), then zI (conjugate) in polar form is:
A) r(cosθ – i sinθ)
B) r(–cosθ + i sinθ)
C) –r(cosθ+i sinθ)
D) r(cos(–θ)+i sinθ)
I A) r(cosθ – i sinθ)
I Conjugate negates imaginary part: zI = r(cosθ – i sinθ) = r·cis(–θ).
Q026.
The number of solutions of z² + |z| = 0 (z complex) is:
A) 1
B) 2
C) 3
D) Infinite
I C) 3
I z=0 is one solution. For z≠0 write z=re^(iθ): r²e^(2iθ)+r=0 → r=1 (→ e^(2iθ)=–1 →
θ=π/2,3π/2). Total: 3.
∑ QUADRATIC EQUATIONS
Q027.
If α, β are roots of 2x²–5x+3=0, then α+β and αβ are:
A) 5/2 and 3/2
B) –5/2 and 3/2
C) 5/2 and –3/2
D) 5 and 3
I A) 5/2 and 3/2
I By Vieta: α+β = –b/a = 5/2; αβ = c/a = 3/2.
Q028.
The discriminant of ax²+bx+c=0 for real and distinct roots must be:
A) ∆ = 0
B) ∆ < 0
C) ∆ > 0
D) ∆ ≥ 0
I C) ∆ > 0
I ∆=b²–4ac. ∆>0: 2 real distinct roots; ∆=0: equal roots; ∆<0: complex roots.
Q029.
The equation whose roots are 2α and 2β (where α,β are roots of x²–5x+6=0) is:
A) x²–10x+24=0
B) x²–5x+12=0
C) x²–10x+12=0
D) x²+10x+24=0
I A) x²–10x+24=0
I α+β=5, αβ=6. New roots: 2α+2β=10, 2α·2β=4·6=24. Equation: x²–10x+24=0.
Q030.
If one root of x²–px+q=0 is twice the other, then:
A) 2p²=9q
B) p²=2q
C) p²=9q
D) 2p²=q
I A) 2p²=9q
I Roots α,2α. α+2α=3α=p→α=p/3. α·2α=2α²=q→2(p/3)²=q→2p²/9=q→2p²=9q.
Q031.
The maximum value of f(x) = –x² + 4x – 3 is:
A) 1
B) 3
C) 4
D) –3
I A) 1
I Vertex at x=–b/2a=–4/(–2)=2. f(2)=–4+8–3=1. For a<0, vertex gives maximum.
Q032.
The number of real roots of x² + x + 1 = 0 is:
A) 0
B) 1
C) 2
D) Infinite
I A) 0
I ∆ = 1–4 = –3 < 0. No real roots. Both roots are complex conjugates.
Q033.
If α+β=–2 and αβ=–3, the quadratic equation is:
A) x²–2x–3=0
B) x²+2x–3=0
C) x²+2x+3=0
D) x²–2x+3=0
I B) x²+2x–3=0
I x²–(α+β)x+αβ=0 → x²–(–2)x+(–3)=0 → x²+2x–3=0.
Q034.
The equation xI–5x²+4=0 has how many real solutions?
A) 0
B) 2
C) 4
D) 1
I C) 4
I Let y=x²: y²–5y+4=0→(y–1)(y–4)=0→y=1,4→x=±1,±2. Four real solutions.

Q035.
The value of ¹ICI is:
A) 120
B) 720
C) 30
D) 240
I A) 120
I ¹ICI = 10!/(3!×7!) = (10×9×8)/(3×2×1) = 720/6 = 120.
Q036.
The number of ways to arrange the letters of the word MISSISSIPPI is:
A) 34650
B) 11!
C) 4320
D) 1260
I A) 34650
I 11 letters: M(1), I(4), S(4), P(2). Ways = 11!/(1!4!4!2!) = 39916800/1152 = 34650.
Q037.
In how many ways can 5 boys and 3 girls be seated in a row so that no two girls are adjacent?
A) 14400
B) 720
C) 4320
D) 2880
I A) 14400
I First arrange 5 boys: 5! = 120 ways. 6 gaps (including ends). Choose 3 of 6 for girls:
6P3=120. Total=120×120=14400.
Q038.
nCr + nCr–1 equals:
A) II¹CI
B) ICIII
C) II¹CIII
D) 2·ICI
I A) II¹CI
I Pascal's identity: ICI + ICIII = II¹CI. Fundamental to Pascal's triangle.
Q039.
The number of diagonals in a polygon of n sides is:
A) n(n–1)/2
B) n(n–3)/2
C) n(n–1)
D) nCI
I B) n(n–3)/2
I Total diagonals = ICI – n = n(n–1)/2 – n = n(n–3)/2. (Subtract n sides from all
connections)
Q040.
How many 4-digit numbers can be formed using digits 1–9 without repetition?
A) 3024
B) 6561
C) 9I
D) 9!
I A) 3024
I 9P4 = 9×8×7×6 = 3024. (Permutation: order matters, no repetition)
Q041.
The number of ways to select a committee of 3 from 10 people is:
A) 720
B) 120
C) 30
D) 360
I B) 120
I ¹ICI = 120 (combination, order doesn't matter in committee selection).
Q042.
The number of circular arrangements of 6 distinct objects is:
A) 6!
B) 5!
C) 6!/6
D) Both B and C
I D) Both B and C
I Circular permutations = (n–1)! = 5! = 120. Also 6!/6 = 720/6 = 120.

I B) ICI xIII yI

B) 10

I C) 2I

Q070.
