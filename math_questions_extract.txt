∑ SETS, RELATIONS & FUNCTIONS
Q001.
If A = {1,2,3,4} and B = {3,4,5,6}, then A ■ B (symmetric difference) is:
A) {1,2,5,6}
B) {3,4}
C) {1,2,3,4,5,6}
D) {}
■ A) {1,2,5,6}
■ A ■ B = (A∪B) – (A∩B) = {1,2,3,4,5,6} – {3,4} = {1,2,5,6}
Q002.
The number of subsets of a set with n elements is:
A) n²
B) 2n
C) 2■
D) n!
■ C) 2■
■ Every element has 2 choices (in or out). Total subsets = 2■. For n=3: 2³=8 subsets.
Q003.
If n(A)=3, n(B)=4 and n(A∩B)=2, then n(A∪B) is:
A) 5
B) 7
C) 9
D) 3
■ A) 5
■ n(A∪B) = n(A)+n(B)–n(A∩B) = 3+4–2 = 5
Q004.
A relation R on set A is called an equivalence relation if it is:
A) Reflexive and symmetric only
B) Symmetric and transitive only
C) Reflexive, symmetric and transitive
D) Anti-symmetric and transitive
■ C) Reflexive, symmetric and
transitive
■ Equivalence relation must satisfy all three: reflexive, symmetric AND transitive.
Q005.
Which of the following functions is one-one (injective)?
A) f(x)=x²
B) f(x)=sinx
C) f(x)=2x+3
D) f(x)=|x|
■ C) f(x)=2x+3
■ f(x)=2x+3 is strictly increasing → one-one. Others fail: f(x)=f(-x) for x² and |x|.
Q006.
The domain of f(x) = √(4–x²) is:
A) (–2, 2)
B) [–2, 2]
C) (0, 2)
D) [0, 4]
■ B) [–2, 2]
■ Need 4–x² ≥ 0 → x² ≤ 4 → –2 ≤ x ≤ 2. Endpoints included (square root of 0 is 0).
Q007.
If f(x) = x² and g(x) = √x, then fog(x) is:
A) x
B) x²
C) √x
D) x■
■ A) x
■ fog(x) = f(g(x)) = f(√x) = (√x)² = x, for x ≥ 0.

Q008.
The inverse function of f(x) = (x–1)/(x+1) is:
A) (x+1)/(x–1)
B) (1+x)/(1–x)
C) (x–1)/(x+1)
D) 1/f(x)
■ B) (1+x)/(1–x)
■ Let y=(x–1)/(x+1). Swap x,y and solve: x(y+1)=y–1 → y=(1+x)/(1–x).
∑ MATRICES & DETERMINANTS
Q009.
If A is a 3×3 matrix and |A| = 5, then |3A| is:
A) 15
B) 45
C) 135
D) 375
■ C) 135
■ |kA| = k■|A| for an n×n matrix. |3A| = 3³×5 = 27×5 = 135.
Q010.
A square matrix A is singular if:
A) |A| ≠ 0
B) |A| = 0
C) A = I
D) A = A■
■ B) |A| = 0
■ Singular matrix: |A|=0, no inverse exists. Non-singular: |A|≠0, inverse exists.
Q011.
If A = [[2,1],[5,3]], then A■¹ is:
A) [[3,–1],[–5,2]]
B) [[3,1],[5,2]]
C) [[–3,1],[5,–2]]
D) [[2,–1],[–5,3]]
■ A) [[3,–1],[–5,2]]
■ |A|=6–5=1. A■¹=(1/|A|)×adj(A). adj(A)=[[3,–1],[–5,2]]. Since |A|=1, A■¹=adj(A).
Q012.
The value of the determinant |1 2 3 / 4 5 6 / 7 8 9| is:
A) 0
B) 6
C) –6
D) 45
■ A) 0
■ R3 = R1+R2 (7=1+4? No). Actually R3–R2=R2–R1 (AP rows). Rows in AP → det=0.
Q013.
If A is symmetric, then A■ equals:
A) –A
B) A
C) A■¹
D) 0
■ B) A
■ Symmetric matrix: A■ = A. Skew-symmetric: A■ = –A.
Q014.
The rank of the zero matrix (all entries = 0) is:
A) 1
B) n
C) 0
D) Undefined
■ C) 0
■ Rank = number of non-zero rows in row echelon form. Zero matrix has no non-zero
rows → rank 0.

Q015.
If AB = BA, then matrices A and B are said to be:
A) Orthogonal
B) Commutative
C) Idempotent
D) Involutory
■ B) Commutative
■ Commutative property of multiplication. Generally AB ≠ BA for matrices.
Q016.
For any square matrix A, A + A■ is always:
A) Skew-symmetric
B) Symmetric
C) Zero matrix
D) Identity matrix
■ B) Symmetric
■ (A+A■)■ = A■+A = A+A■. So it equals its own transpose → symmetric.
Q017.
The cofactor C■■ of matrix A = [[1,2],[3,4]] is:
A) 3
B) –3
C) 4
D) –4
■ B) –3
■ C■■ = (–1)^(1+2) × M■■ = (–1)³ × 3 = –3 (minor M■■ is the element 3).
Q018.
If A² = A, the matrix A is called:
A) Involutory
B) Nilpotent
C) Idempotent
D) Orthogonal
■ C) Idempotent
■ A² = A → idempotent. A² = I → involutory. A■ = 0 → nilpotent. AA■ = I → orthogonal.

D) –i
■ D) –i

Q022.
The conjugate of z = (2+3i)/(1–i) is:
A) (–1+5i)/2
B) (–1–5i)/2
C) (1–5i)/2
D) (1+5i)/2
■ B) (–1–5i)/2
■ Multiply num & denom by (1+i): (2+3i)(1+i)/2 = (2+2i+3i–3)/2 = (–1+5i)/2. Conjugate:
(–1–5i)/2.
Q023.
De Moivre's theorem: (cosθ + i sinθ)■ equals:
A) cos(nθ) + i sin(nθ)
B) n cosθ + n i sinθ
C) cos(θ/n)+i sin(θ/n)
D) cos■θ + i sin■θ
■ A) cos(nθ) + i sin(nθ)
■ De Moivre's: (cisθ)■ = cis(nθ) = cos(nθ)+i sin(nθ). Fundamental to complex number
theory.
Q024.
The cube roots of unity are 1, ω, ω². Then 1 + ω + ω² equals:
A) 1
B) 3
C) 0
D) –1
■ C) 0
■ Sum of nth roots of unity = 0. So 1+ω+ω²=0. Also ω³=1 and ω≠1.
Q025.
If z = r(cosθ + i sinθ), then z■ (conjugate) in polar form is:
A) r(cosθ – i sinθ)
B) r(–cosθ + i sinθ)
C) –r(cosθ+i sinθ)
D) r(cos(–θ)+i sinθ)
■ A) r(cosθ – i sinθ)
■ Conjugate negates imaginary part: z■ = r(cosθ – i sinθ) = r·cis(–θ).
Q026.
The number of solutions of z² + |z| = 0 (z complex) is:
A) 1
B) 2
C) 3
D) Infinite
■ C) 3
■ z=0 is one solution. For z≠0 write z=re^(iθ): r²e^(2iθ)+r=0 → r=1 (→ e^(2iθ)=–1 →
θ=π/2,3π/2). Total: 3.
∑ QUADRATIC EQUATIONS
Q027.
If α, β are roots of 2x²–5x+3=0, then α+β and αβ are:
A) 5/2 and 3/2
B) –5/2 and 3/2
C) 5/2 and –3/2
D) 5 and 3
■ A) 5/2 and 3/2
■ By Vieta: α+β = –b/a = 5/2; αβ = c/a = 3/2.
Q028.
The discriminant of ax²+bx+c=0 for real and distinct roots must be:
A) Δ = 0
B) Δ < 0
C) Δ > 0
D) Δ ≥ 0
■ C) Δ > 0
■ Δ=b²–4ac. Δ>0: 2 real distinct roots; Δ=0: equal roots; Δ<0: complex roots.

Q029.
The equation whose roots are 2α and 2β (where α,β are roots of x²–5x+6=0) is:
A) x²–10x+24=0
B) x²–5x+12=0
C) x²–10x+12=0
D) x²+10x+24=0
■ A) x²–10x+24=0
■ α+β=5, αβ=6. New roots: 2α+2β=10, 2α·2β=4·6=24. Equation: x²–10x+24=0.
Q030.
If one root of x²–px+q=0 is twice the other, then:
A) 2p²=9q
B) p²=2q
C) p²=9q
D) 2p²=q
■ A) 2p²=9q
■ Roots α,2α. α+2α=3α=p→α=p/3. α·2α=2α²=q→2(p/3)²=q→2p²/9=q→2p²=9q.
Q031.
The maximum value of f(x) = –x² + 4x – 3 is:
A) 1
B) 3
C) 4
D) –3
■ A) 1
■ Vertex at x=–b/2a=–4/(–2)=2. f(2)=–4+8–3=1. For a<0, vertex gives maximum.
Q032.
The number of real roots of x² + x + 1 = 0 is:
A) 0
B) 1
C) 2
D) Infinite
■ A) 0
■ Δ = 1–4 = –3 < 0. No real roots. Both roots are complex conjugates.
Q033.
If α+β=–2 and αβ=–3, the quadratic equation is:
A) x²–2x–3=0
B) x²+2x–3=0
C) x²+2x+3=0
D) x²–2x+3=0
■ B) x²+2x–3=0
■ x²–(α+β)x+αβ=0 → x²–(–2)x+(–3)=0 → x²+2x–3=0.
Q034.
The equation x■–5x²+4=0 has how many real solutions?
A) 0
B) 2
C) 4
D) 1
■ C) 4
■ Let y=x²: y²–5y+4=0→(y–1)(y–4)=0→y=1,4→x=±1,±2. Four real solutions.

C) 4320

■ A) 34650

Q037.
In how many ways can 5 boys and 3 girls be seated in a row so that no two girls are adjacent?
A) 14400
B) 720
C) 4320
D) 2880
■ A) 14400

Q038.

■ A) ■■¹C■

Q039.

■ Rolle's theorem: if f(a)=f(b), then ∃c∈(a,b) s.t. f'(c)=0. (Special case of MVT)

■ B) f'(c)=0 and f''(c)>0
